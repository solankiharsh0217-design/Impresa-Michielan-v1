# Impresa Edile Michielan - Variant 2

This is the second variant of the website for **Impresa Edile Michielan**, a family-run construction company based in Scorzè, Venice.

## Design System: Warm Organic Craftsmanship

- **Concept**: Storytelling + Trust (narrative-driven layout with emphasis on heritage and craftsmanship).
- **Style**: Editorial feel with rich textures, warm overlays, and photocentric layouts.
- **Colors**: 
  - Primary: `#1D1348`
  - Secondary: `#E31611`
  - Background: `#F5F0EB`
- **Typography**: Playfair Display (Serif) & Montserrat (Sans-serif).

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS / Tailwind (for layout utils)
- **Icons**: Lucide React
- **Animations**: Framer Motion (basic) / CSS Transitions

## Features

- **Multi-page Architecture**: Home, Services, About, Opere (Gallery), Contact.
- **Service Modals**: Detailed service descriptions and galleries within the Services page.
- **Storytelling Layout**: Narrative approach to brand history and philosophy.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Auto-scroll to Top**: Automatic scroll to top on every page navigation.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
